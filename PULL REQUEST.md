#  COMUNICHAIN 
## Contrato simbólico del equipo para el desarrollo del proyecto

---

## 1. Nombre del Proyecto  
**Comunichain**  
Plataforma descentralizada que gestiona y transparenta cómo las comunidades y gobiernos utilizan el presupuesto asignado, garantizando trazabilidad y transparencia mediante tecnología distribuida.

---

## 2. Alcance del Proyecto  
El equipo se compromete a desarrollar una plataforma que permita:

- Visualizar presupuestos asignados a una comunidad.  
- Dar seguimiento al uso de presupuesto de manera transparente.  
- Mostrar trazabilidad de cada movimiento registrado.  
- Asegurar que la información no pueda ser manipulada gracias a la descentralización.  
- Entregar prototipos, documentación y avances conforme los plazos establecidos.

---

## 3. Entregables  

1. Documento del proyecto (visión, alcance, arquitectura).  
2. Contrato simbólico en GitHub mediante pull request aprobada por todos.  
3. Prototipo funcional o maqueta de la plataforma Comunichain.  
4. Presentación final del proyecto con flujos, ventajas y modelo de valor.  
5. Repositorio organizado con commits y responsabilidades claras.
